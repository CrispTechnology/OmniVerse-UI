## 👨‍💻 Dev Setup

```bash
git clone https://github.com/badboysm890/ClaraVerse.git
cd ClaraVerse
npm install
npm run dev           # For web
npm run electron:dev  # For desktop
```

### ⚒️ Build

```bash
npm run build              # Web
npm run electron:build     # Desktop
```
